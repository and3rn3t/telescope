Usage Guidelines

✓ Do

  • • Use cosmic violet for primary actions
  • • Maintain 4.5:1 contrast ratios
  • • Apply consistent spacing scales
  • • Use monospace fonts for data values
  • • Add subtle hover animations

✗ Don't

  • • Use bright colors that compete with imagery
  • • Apply inconsistent border radius
  • • Mix different spacing scales
  • • Use color alone to convey information
  • • Create jarring animations
